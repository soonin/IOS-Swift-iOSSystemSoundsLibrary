# IOS-Swift-iOSSystemSoundsLibrary
``` Level : Intermediate ```

This tutorial shows how to make an UITableView of all available iOS system's sound and play them by tapping on them to quickly find the best for your app.Yes, There are some predefined system sounds, for the system sound ID in the range 1000 to 2000 (decimal), as shown below (from 2.0 to 5.0 beta). The system sounds are all stored in /System/Library/Audio/UISounds/. This tutorial is made with Xcode 10 and built for iOS 12.

## About

Sample app to Make TableView of all iOS system sounds IDs ( tap and play Sounds ) 

## Requirements

- iOS 12.0+
- Xcode 10.0+
- Swift 4.0+

## Usage

You won't need to Download any extra libaray just ```import UIKit``` inside the project
